# Proto - Labour Management System

Welcome to the Proto monorepo. This project is a comprehensive Labour Management System built with React Native (Expo) and Node.js.

## **Documentation Suite**

Detailed documentation is available in the `docs/` directory:

1.  [**Project Overview**](./docs/01_PROJECT_OVERVIEW.md)
    *   Setup instructions, audience, and core purpose.
2.  [**Architecture & Tech Stack**](./docs/02_ARCHITECTURE.md)
    *   System design, folder structure, and data flow diagrams.
3.  [**API & Database Schema**](./docs/03_API_AND_SCHEMA.md)
    *   Full database schema (SQLite) and API endpoint reference.
4.  [**Features & Logic**](./docs/04_FEATURES_AND_LOGIC.md)
    *   Deep dive into wage calculation algorithms, attendance locking, and error handling.

## **Quick Start**

### **Server**
```bash
cd server
npm install
node index.js
```

### **Client**
```bash
cd client
npm install
npx expo start
```
